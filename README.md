# 🏗️ Flipping-App Pro: Real Estate Operations Control

![Status](https://img.shields.io/badge/Status-Project%20Pivot-orange)
![Sector](https://img.shields.io/badge/Sector-PropTech-blue)
![IA](https://img.shields.io/badge/Features-OCR%20Integration-gold)

**Flipping-App Pro** es una plataforma de gestión integral diseñada para centralizar el ciclo de vida de inversiones inmobiliarias de compra, reforma y venta (Flipping). 

El objetivo principal es mitigar el riesgo financiero mediante el control estricto de costos en tiempo real, permitiendo la transparencia total entre el **Socio Gestor** (Director de Obra) y el **Socio Inversor**.

---

## 🎯 Objetivos del Sistema

* **Control Presupuestario:** Comparativa automática entre Inversión Estimada vs. Gasto Real (Variance Analysis).
* **Digitalización de Comprobantes:** Captura de facturas y tickets mediante **OCR** para automatizar el ingreso de datos.
* **Perfiles Diferenciados:** * **Gestor:** Carga operativa, gestión de proveedores y control de obra.
    * **Inversor:** Dashboard de KPIs, salud financiera del proyecto y proyección de ROI.
* **Auditoría:** Historial centralizado de todos los movimientos de caja de la obra.

## 🛠️ Stack Tecnológico

- **Lenguaje:** Python 3.x
- **Framework UI:** Streamlit (Prototipado de alta fidelidad)
- **Motor de IA:** OCR Engine (Tesseract / Google Vision)
- **Persistencia:** SQLite / PostgreSQL
- **Metodología:** SDLC Ágil con enfoque en Integridad de Datos.
